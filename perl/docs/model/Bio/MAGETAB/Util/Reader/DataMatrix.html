NAME

Bio::MAGETAB::Util::Reader::DataMatrix - Data matrix parser class.

SYNOPSIS

 use Bio::MAGETAB::Util::Reader::DataMatrix;
 my $parser = Bio::MAGETAB::Util::Reader::DataMatrix->new({
     uri => $dm_filename,
 });
 my $data_matrix = $parser->parse();

DESCRIPTION

This class is used to parse data matrix files. It can be used on its own, but more often you will want to use the main Bio::MAGETAB::Util::Reader class which handles extended parsing options more transparently.

ATTRIBUTES

See the Bio::MAGETAB::Util::Reader::TabFile manpage for superclass attributes.

magetab_object

A Bio::MAGETAB::DataMatrix object. This can either be set upon instantiation, or a new object will be created for you. It can be retrieved at any time using get_magetab_object.

METHODS

parse

Parse the data matrix pointed to by $self->get_uri(). Returns the Bio::MAGETAB::DataMatrix object updated with the data matrix contents.

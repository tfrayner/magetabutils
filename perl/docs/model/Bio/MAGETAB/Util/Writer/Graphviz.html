NAME

Bio::MAGETAB::Util::Writer::Graphviz - Visualization of MAGE-TAB objects.

SYNOPSIS

 use Bio::MAGETAB::Util::Writer::Graphviz;
 my $drawer = Bio::MAGETAB::Util::Writer::Graphviz->new({
    magetab    => $magetab_container,
    filehandle => $output_fh,
    font       => 'luxisr',
 });
 
 $drawer->draw();

DESCRIPTION

This is a simple visualization class for MAGE-TAB objects. It may be developed further in future; at the moment, given a Bio::MAGETAB container and a filehandle, it will just generate a '.dot' file which can then be processed using the Graphviz ``dot'' application.

ATTRIBUTES

magetab

The Bio::MAGETAB container containing the MAGE-TAB objects to visualize. This is a required attribute. See the Bio::MAGETAB manpage for more information on this container class.

filehandle

The filehandle to use to output (i.e. the ``dot'' input file).

font

The font used for object labels in the output.

METHODS

draw

Generates output suitable for use with the Graphviz ``dot'' application.

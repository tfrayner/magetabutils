NAME

Bio::MAGETAB::Extract - MAGE-TAB extract class

SYNOPSIS

 use Bio::MAGETAB::Extract;

DESCRIPTION

This class is used to store information on biological extracts (e.g., RNA, DNA) in MAGE-TAB. the Bio::MAGETAB::Material manpage for superclass methods.

ATTRIBUTES

No class-specific attributes. See the Bio::MAGETAB::Material manpage.

METHODS

No class-specific methods. See the Bio::MAGETAB::Material manpage.
